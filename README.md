# Pygame-snake-pro
Retro Snake Game built with Python &amp; Pygame | Levels, speed scaling, and smooth rounded graphics

### Features
- 24x24 Grid System
- Level & Speed Scaling
- Rounded UI Graphics
- Dual Control Scheme
- Collision Detection
- Score + High Score System
- Game States / UI Flow
- Food Spawn Logic

### Technology Used 
- python
- pygame
