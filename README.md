# Pygame-snake-pro
Retro Snake Game built with Python &amp; Pygame | Levels, speed scaling, and smooth rounded graphics

### Features
- 24x24 Grid System
- Level & Speed Scaling
- Rounded UI Graphics
- Dual Control Scheme
- Collision Detection
- Score + High Score System
- Game States / UI Flow
- Food Spawn Logic

### Technology Used 
- python
- pygame

### Python Concepts Used
- Functions
- Lists & Tuples  
- Loops
- Conditionals
- Tuples as Coordinates
- Variables & Constants
- List Methods
- Random Module
- Membership Testing
- Arithmetic & Logic
- F-Strings
- Entry Point Guard
- Return Values

### Purpose
modern take on the classic Snake arcade game built with Python and Pygame. It’s designed to demonstrate clean game logic, level progression, and polished UI in a single-file project. The game gets faster and scores higher as you level up, with rounded graphics, dual controls, and a high-score system to make it feel more complete than the basic tutorial version
