Retro Snake Game

Retro Snake Game built with Python & Pygame | Levels, speed scaling, and smooth rounded graphics

A modern take on the classic Snake game made with Python and Pygame. Navigate a 24x24 grid, eat the red apples to grow, and avoid hitting walls or yourself. The game gets faster as you level up every 50 points, keeping the challenge fresh.

Features
Level system: Speed increases every 50 points, up to 20 FPS 
Smooth UI: Rounded snake segments, faint grid, and clean HUD
Dual controls: Play with Arrow keys or WASD 
Score tracking: High score saved for the current session with a game-over screen
Clean visuals: Dark green theme with distinct head, body, and food colors

How to Run
pip install pygame
python snake.py
Controls: `W/A/S/D` or `Arrow Keys` to move | `SPACE` to retry | `ESC` to quit
